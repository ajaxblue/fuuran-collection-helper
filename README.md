# fuuran-collection-helper

## Purpose

Fuuran Collection Helper is a private internal-use application for my own Etsy workflow. It connects only to my authorized Etsy account to read shop and listing data for personal inventory tracking, pricing comparison, and collection research for orchid-related items. It is not public, not customer-facing, not commercial, and not offered to third parties.

It is:
- single-user only
- not public-facing
- not a commercial service
- not offered to third parties
- intended only to access the operator's own authorized Etsy data
