# fuuran-collection-helper

## Purpose

Fuuran Collection Helper is a private internal-use tool for personal Etsy-related reporting and analysis. It is used only by the operator to read authorized Etsy shop and listing data for inventory tracking, pricing comparison, and collection research for orchid-related items. It is not public-facing, not a commercial service, and not intended for third-party use.

It is:
- single-user only
- not public-facing
- not a commercial service
- not offered to third parties
- intended only to access the operator's own authorized Etsy data
